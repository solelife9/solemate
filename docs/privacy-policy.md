## 1. 수집하는 개인정보 항목

| 항목 | 수집 시점 | 목적 |
|---|---|---|
| **위치 정보(GPS 좌표·경로)** | 러닝 추적 중(포그라운드/백그라운드) | 거리·페이스·경로 기록 |
| **동작 및 피트니스(걸음 수)** | 러닝 중 | 케이던스(분당 걸음 수) 측정 |
| **사진(선택)** | 사용자가 신발 사진 등록 시 | 러닝화 카드 이미지 |
| **러닝 기록**(거리·시간·페이스·신발·메모) | 러닝 저장 시 | 기록 보관·통계·신발 수명 추적 |
| **기기 식별자 / 익명 계정 ID** | 앱 최초 실행 | 기록 동기화(익명) |
| **푸시 토큰(FCM)** | 알림 허용 시 | 교체 시기·리텐션 알림 |

- 이름·이메일 등 직접적 신원정보는 **소셜 로그인(카카오·네이버·Google)을 사용할 때만** 해당 제공자를 통해 최소 수집(닉네임 등)합니다. 이메일 회원가입을 쓰지 않으면 익명으로 동작합니다.

## 2. 이용 목적
- 러닝 기록·통계 제공, 러닝화 수명/교체 시점 추적, 랭킹·업적, 교체/리텐션 알림.

## 3. 보관 및 파기
- 기록은 사용자가 삭제하거나 계정을 삭제할 때까지 보관하며, 요청 시 지체 없이 파기합니다.
- 기기 내 로컬 저장(AsyncStorage)과 서버(동기화 시)에 보관됩니다.

## 4. 제3자 제공 및 처리위탁
- **Google Firebase**(인증·푸시·분석/크래시 리포트): 익명 ID·푸시 토큰·크래시 로그 처리.
- **지도/역지오코딩**: 좌표→지역명 변환(OpenStreetMap Nominatim), 지도 표시(Google Maps).
- 위 사업자 외 제3자에게 개인정보를 판매하지 않습니다.

## 5. 권한 안내(거부 시 영향)
- **위치**: 거부 시 러닝 거리·경로 기록 불가. **항상 허용**은 화면을 꺼도 기록하기 위함.
- **동작·피트니스**: 거부 시 케이던스 측정 불가(다른 기능은 정상).
- **사진**: 거부 시 신발 사진만 등록 불가.
- **알림**: 거부 시 교체/리텐션 알림 미수신.
